# 🔥 Discipline Bot

Telegram-бот для контроля над собой, привычками и целями.

## 📦 Установка

### 1. Клонируй/скопируй файлы проекта

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Получи токен бота
1. Открой Telegram → найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot`
3. Дай имя: `Discipline Bot`
4. Дай username: `my_discipline_bot` (уникальный)
5. Скопируй токен

### 4. Вставь токен

Открой файл `config.py`:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```

Или создай `.env` файл:
```
BOT_TOKEN=ВАШ_ТОКЕН_ЗДЕСЬ
```

### 5. Запусти бота
```bash
python bot.py
```

## ⚙️ Функции бота

| Кнопка | Что делает |
|--------|-----------|
| ✅ Сегодня чисто | Отмечает день без срыва, считает серию |
| 💥 Срыв | Сбрасывает серию (с подтверждением) |
| 🎯 Мои цели | Добавить/отметить задачи на день |
| ⚡ Челлендж дня | Случайный или по сложности челлендж |
| 📊 Статистика | Серия, уровень, прогресс за неделю |
| 💬 Мотивация | Умная мотивация по ситуации |
| 🚨 Мне сейчас тяжело | Жёсткий стоп + срочное действие |
| ⚙️ Настройки | Время подъёма/сна, напоминания |

## 🌐 Деплой на сервер (необязательно)

### Вариант 1: Railway.app (бесплатно)
1. Загрузи код на GitHub
2. Зайди на railway.app
3. New Project → GitHub repo
4. Добавь переменную `BOT_TOKEN` в Environment

### Вариант 2: VPS (Ubuntu)
```bash
# Установка
sudo apt update && sudo apt install python3-pip
pip3 install -r requirements.txt

# Запуск в фоне
nohup python3 bot.py &

# Или через systemd (рекомендуется)
```
